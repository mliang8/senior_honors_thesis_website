HTML Content Elements

Inline Elements

Text markup

Within text flow, you can emphasize, strongly emphasize, mark deletions, use superscript or subscript. You can also mark keyboard input, code snippets, small print. Citing Sources or inline quotes are also supported.

Abbreviations can be marked using the abbr element. You can highlight keywords or special terms and temporarily mark text passages.

Data markup

For machine processing, temporal or generic data can be annotated.

Text structure

Paragraph. A paragraph contains a paragraph of text. Inside the paragraph, only inline elements can be used, e.g. to emphasize words or sentences.

Next paragraph. Like in any longer text, paragraphs are used to separate lines of thoughts. Paragraphs are usually rendered by separating them visually by a vertical margin.

A block quote can be used to quote a full sentence or paragraph from a different source. It is usually rendered with a left margin/indentation.
